# rct-inforatio-analysis
R scripts for the pre-specified statistical analysis of the multicenter RCT evaluating the Inforatio technique for diabetic foot ulcers (DFUs). Shared before data export from RedCap and therefore also before execution to support transparency and document blinded, predefined analysis.

# Link to published protocol
Protocol published in BMJ Open: https://bmjopen.bmj.com/content/12/9/e062344. 
The Statistical analysis plan and the Handling of Missing Data section describe the predefined statistical analysis plan, which is followed in the R scripts uploaded in this Github project.
